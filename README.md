# beancount importer

beancount importer is a GUI tool for importing Alipay/Wechat (and maybe more) bill to beancount file.

## Prerequisite

- Install Python 3 (>= 3.6)

```
pip install beancount PyQt5
```

## Run

```
python3 main.py
```
